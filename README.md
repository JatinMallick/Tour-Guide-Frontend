
# Tour Project

Tour Project is a small, static demo website that showcases a travel/tour agency landing page. It demonstrates a clean layout, an image gallery, and simple interactive behavior using vanilla HTML, CSS, and JavaScript.

## Features

- Responsive landing page layout
- Image gallery for destinations
- Simple interactive elements (carousel, hover effects)
- Placeholder contact section for inquiries

## Technologies

- HTML5
- CSS3
- JavaScript 

## Project Structure

- index.html
- css/
  - style.css
- images/
- js/
  - script.js

## Getting Started

1. Open [index.html](index.html) in your browser to view the site locally.
2. (Optional) Serve the project with a simple HTTP server for local development:

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server -p 8000
```

Then open http://localhost:8000 in your browser.
